# 🖥️ Proyecto Linux – Simulación de Entorno Empresarial

## 📌 Descripción

Este proyecto consiste en la simulación de un entorno empresarial en Linux, donde se gestionan directorios, archivos y permisos para representar diferentes departamentos.

---

## 🧩 Tecnologías utilizadas

* Linux (Ubuntu – terminal)
* mkdir, cd, touch, chmod, ls -l, cat, sudo

---

## ⚙️ Funcionalidades

* Simulación de departamentos: Administración, Ventas y TI
* Creación de directorios
* Gestión de archivos
* Configuración de permisos (chmod)
* Uso de sudo para administración

---

## 📸 Evidencia

Las evidencias están incluidas en el documento PDF de documentación adjunto en el repositorio.

---

## 🧠 Aprendizaje

Se aprendió a gestionar archivos, permisos y control de acceso en Linux, simulando un entorno real de administración de sistemas.
